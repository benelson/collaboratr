# collaboratr
Connecting people with expertise to those who want to learn or collaborate.

## Requirements
* networkx-1.1
* nxpd-0.2.0

## Installation
If you get an error when trying to draw graphs like `dyld: Library not loaded: /usr/lib/libltdl.7.dylib`, try
```
$> brew install libtool
$> brew link libtool
```
